**BetonQuestAddon** is an essential third-party extension designed to push the boundaries of the **BetonQuest** plugin. It fills the gaps in native functionality by introducing specialized features that allow for more complex and immersive quest-making.

## ✨ What is BetonQuestAddon?

BetonQuestAddon acts as a bridge between BetonQuest and other popular plugins, expanding your creative toolkit. It adds a wide array of custom:

* **Actions:** Trigger new behaviors and rewards.
* **Objectives:** Create diverse tasks for players to complete.
* **Conditions:** Check for specific states or requirements that were previously unsupported.

## 🚀 Why Use This Addon?

If you are a server owner or quest creator who feels limited by BetonQuest's native integration capabilities, this plugin is for you.

* **Bridge the Gap:** Easily integrate with external plugins like `Nexo`, `CustomCrops`, and `CustomFishing` without complex workarounds.
* **Enhanced Interactivity:** Build quests that react to cosmetic changes, custom farming activities, or unique item interactions.
* **Streamlined Development:** Save time by using pre-built integrations for the most popular modern Spigot/Paper plugins.

## 🔌 Supported Integrations

Expand your quest system with deep support for:

## ⚠️ Important Information

* **Version Requirement:** BetonQuestAddon is built specifically for the **latest version of BetonQuest**. To ensure stability and access to all features, please keep your BetonQuest installation up to date.
* **Dependency:** This is a third-party addon. You must have [BetonQuest](https://betonquest.org) installed on your server for this plugin to function.
